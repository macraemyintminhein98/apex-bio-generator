# BioBoost AI

Elevate your Instagram game with BioBoost AI! This web application generates creative and catchy Instagram bios tailored to your keywords.

## Features:
- **AI-Powered Generation:** Input keywords (e.g., 'travel blogger', 'fitness enthusiast') and receive diverse bio suggestions.
- **Quick & Easy:** Get multiple options in seconds.
- **Copy-to-Clipboard:** Easily copy your favorite bio with a single click.
- **Mobile-First Design:** Optimized for a seamless experience on any device.

## How to Use:
1. Enter keywords that describe you or your brand into the input field.
2. Click 'Generate Bios'.
3. Browse the AI-generated options.
4. Click 'Copy' next to your preferred bio to add it to your clipboard.

## Development Status:
This project is a rapid prototype for a digital product. The `index.html` showcases the user interface and simulates the AI generation and copy functionality. For a full product, it would require backend integration (e.g., Vercel Serverless Function + OpenAI API) and payment processing (e.g., Stripe).

## Future Enhancements:
